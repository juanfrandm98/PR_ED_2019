Descripción detallada

T.D.A. Diccionario.

Una instancia del tipo de datos abstracto Diccionario es un objeto con un conjunto de strings. Dicho conjunto almacena una serie de strings no repetidos.

El TDA proporciona una serie de operaciones para trabajar con los strings y con sus letras, así como un iterador para recorrer el conjunto.

Juan Francisco Díaz Moreno
