Una instancia {\itshape c} del tipo de datos abstracto {\ttfamily \doxyref{Juego}{p.}{classJuego}} es un objeto con 2 campos\+: \begin{DoxyVerb}- Un JuegoLetras que permite jugar partidas de Letras.
- Un JuegoCifras que permite jugar partidas de Cifras.
\end{DoxyVerb}
