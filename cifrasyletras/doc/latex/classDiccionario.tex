\subsection{Descripción detallada}
T.\+D.\+A. \doxyref{Diccionario}{p.}{classDiccionario}. 

Una instancia {\itshape del} tipo de datos abstracto {\ttfamily \doxyref{Diccionario}{p.}{classDiccionario}} es un objeto con un conjunto de strings. Dicho conjunto almacena una serie de strings no repetidos.

El T\+DA proporciona una serie de operaciones para trabajar con los strings y con sus letras, así como un iterador para recorrer el conjunto.
