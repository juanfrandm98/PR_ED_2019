\subsection{Descripción detallada}
T.\+D.\+A. \doxyref{Juego\+Letras}{p.}{classJuegoLetras}. 

Una instancia {\itshape c} del tipo de datos abstracto {\ttfamily \doxyref{Juego\+Letras}{p.}{classJuegoLetras}} es un objeto con 3 campos\+: \begin{DoxyVerb}- Un diccionario que incluye las palabras que se podrán adivinar en
  la partida de Letras.
- Una bolsa de letras que incluye las letras disponibles inicialmente
  y la mano de juego actual.
- Un conjunto de letras que incluye la información de las letras en
- juego.
\end{DoxyVerb}


Permite jugar partidas de Letras.
