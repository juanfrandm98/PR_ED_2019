Descripción detallada

TDA Mi_Fecha_Historica.

Una instancia c del tipo de datos abstracto Mi_Fecha_Historica es un objeto con cuatro campos: tres enteros y un vestor de Strings.

Uno de los enteros es la fecha base del conjunto de eventos y los otros dos son la base del vector dinámico que representamos como:

Año Número de eventos Evento/s:

  • Evento_1
  • Evento_2
  • ...
  • Evento_N
